# GitHub Profile Finder

A clean, responsive web application that fetches and displays real-time GitHub user profile data using the official GitHub REST API.

## Features
* Fetch real-time data (Avatar, Name, Bio, Followers, Following, and Public Repositories).
* Handles edge cases (Displays an error message if a user is not found).
* Fully accessible via keyboard (Pressing `Enter` triggers the search).
* Modeled with a dark theme inspired by GitHub's native UI.

## Technologies Used
* HTML5
* CSS3 (Flexbox, Custom Dark Theme)
* JavaScript (Async/Await, Fetch API, DOM Manipulation)

## How to Run Locally
1. Clone this repository or download the source code.
2. Open `index.html` directly in any modern web browser.
